Propósito del Sitio

Conocer los diferentes estilos que podemos aplicar a los elementos que se encuentran en las páginas web, para que estas sean más llamativas. También se busca que las personas que visiten el sitio web, puedan adquirir conocimientos básicos en las propiedades y valores que se aplicarán a la etiqueta que hara más atractiva visualmente las páginas web.

Hojas de Estilo en cascada CSS

Para empezar vamos a conocer lo que significan las siglas CSS: (Cascading Style Sheets u hojas de estilo en cascada), es un lenguaje de programación que permite aplicar estilos, de modo que los títulos, listas y párrafos se vean igual en todas y cada una de las páginas web. Estos estilos son bastante útiles para ahorrar tiempo y para controlar el diseño de varias páginas web a la vez haciendo que las páginas se puedan visualizar en los diferentes dispositivos y tamaños de pantalla.

Sintaxis básica de CSS

Se compone de tres partes:

  1. Selector: indica a que etiqueta HTML se aplicará el estilo.
  2. Propiedad: indica a que propiedad de la etiqueta se aplicará el estilo, como el tamaño de la letra, tipo de letra, color de fondo, imagénes y color de texto.
  3. Valor: valor de la propiedad

A continuación veremos un ejemplo, de como se puede usar los estilos de CSS, en la imagen de la izquierda observamos que la etiqueta style se coloca dentro del archivo HTML para dar el estilo que requerimos. En la imagen de la derecha observamos un archivo CSS externo, que contiene las reglas de estilos en lenguaje CSS, el cuál funciona insertando un link en el encabezado de los archivos de HTML de nuestras páginas.

Frameworks CSS

Es una biblioteca de hojas de estilo establecidas para ser utilizadas como base en el diseño o maquetación de páginas web y son bastante útiles a la hora de optimizar tiempo, cumple con normas estándar y funciona en la mayoría de los navegadores. Algunos de los Frameworks más utilizados son los siguientes:

  • Awsm
  • Armazon
  • Baseguide
  • Bootstrap
  • BoxySheets
